DeducerRichOutput contains a number of packages designed to be used with the Deducer package:

 - RichOutput: This alters the console to display HTML formatting and provide an output navigation pane.
 - Alpha:  A dialog for computing Cronbach's alpha, reverse-scoring variables, and creating a new mean from a set of variables.
 - ANOVA:  A dialog for the ezANOVA function in package 'ez'.
 - Reshape:  A dialog for reshaping data from wide to long or long to wide format, generally in preparation for within-subjects ANOVA.
